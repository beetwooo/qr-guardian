# Project Overview

A minimal static HTML/CSS webpage.

## How to Run

The workflow `Start application` serves the site with Python's built-in HTTP server:

```
python3 -m http.server 5000
```

Open the preview pane to see the page.

## Stack

- Plain HTML5 (`index.html`)
- Plain CSS3 (`style.css`)
- No build tools or dependencies
